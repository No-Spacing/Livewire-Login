# Laravel-Livewire
 * A simple login with register page.
 * It uses Laravel and Livewire.
# Setting up
 * Open your command terminal inside your project folder and type the following:
   ```
   composer install
   cp .env.example .env
   php artisan key:generate
   php artisan serve
   ```
